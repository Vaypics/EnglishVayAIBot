# EnglishVay - Практика английского через AI-ролеплей

**EnglishVay** - это продвинутый Telegram-бот, который превращает изучение английского в живое общение с персонажами. Вместо скучных тестов ты выстраиваешь отношения с AI-героями, у каждого из которых свой характер, сленг и память о ваших прошлых беседах.

Проект реализован на связке **n8n** (логика), **Supabase** (БД и векторный поиск) и **OpenAI** (мозги).

---

## Технологический стек

* **Automation:** n8n (сложные цепочки логики и агентские воркфлоу)
* **AI Models (OpenAI):**
    * `gpt-4o-mini`: Основная логика и чат.
    * `gpt-4o-mini (Vision)`: Анализ изображений.
    * `Whisper`: Преобразование голоса пользователя в текст.
    * `TTS (Text-to-Speech)`: Озвучка ответов персонажей.
* **Database:** Supabase (PostgreSQL) + расширение `pgvector` для долговременной памяти.
* **External Integration:** Tavily API (поиск в реальном времени), Telegram Stars (оплата).

## Ключевые возможности

### 1. Матрица отношений (Relationship Matrix)
Система динамически отслеживает два параметра для каждого персонажа: **Friendship** и **Romance** (0-100). Твои ответы напрямую влияют на то, кем ты станешь для героя: лучшим другом, романтическим партнером или заклятым врагом. Это меняет тон общения и доступный контент.

### 2. Auditor AI - твой строгий тренер
Каждое сообщение пользователя проходит через скрытого «аудитора». Он не просто исправляет ошибки, а дает фидбек в формате:
* **Mistakes:** Разбор грамматики.
* **English Pros:** Похвала за использование крутых оборотов.
* **XP System:** Опыт начисляется только за грамотные и содержательные ответы.

### 3. Мультимодальный опыт
* **Голос:** Общайся аудиосообщениями - персонажи понимают тебя и отвечают своим уникальным голосом.
* **Зрение:** Присылай фото (еда, обстановка, документы), и персонажи обсудят их с тобой, используя актуальную лексику.

### 4. Автономность
Благодаря планировщику (Scheduler), персонажи могут сами «вспомнить» о тебе и написать первыми, инициируя диалог на основе их текущих дел или вашего общего контекста.

## Как развернуть проект

### 1. База данных
Выполни SQL-запрос из файла `database/database_schema.sql` в консоли **Supabase SQL Editor**. Это создаст таблицы:
* `users`: Профили, очки отношений и статус оплаты.
* `personas`: Промпты и настройки голоса.
* `chat_logs`: История сообщений для памяти.
* `documents`: Векторное хранилище знаний LangChain.

### 2. Воркфлоу n8n
Импортируй файлы из папки `/workflows` в свой экземпляр n8n:
1.  `main_bot_logic.json`: Ядро и обработка сообщений.
2.  `event_scheduler.json`: Система рассылок.
3.  `payment_system.json`: Обработка платежей через Telegram Stars.

### 3. Настройка окружения
В n8n создай Credentials для:
* **Telegram API** (токен от @BotFather)
* **OpenAI API**
* **Supabase API**
* **Tavily API**
